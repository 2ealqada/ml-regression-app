# ML Regression App

Eine kleine Web-App zum Vergleich von Random Forest und k-Nearest Neighbor für Regressionsaufgaben.

## Ziel des Projekts

Diese App zeigt, wie Machine-Learning-Modelle für eine Regressionsaufgabe trainiert und bewertet werden können.

Das Projekt basiert auf meiner Bachelorarbeit zum Vergleich von Random Forest und kNN.

## Funktionen

- Beispieldatensatz anzeigen
- Train/Test Split durchführen
- Random Forest Modell trainieren
- kNN Modell trainieren
- MAE, RMSE und R² anzeigen
- Prediction vs. Real Diagramm anzeigen
- Feature Importance für Random Forest anzeigen

## Technologien

- Python
- Streamlit
- Pandas
- scikit-learn
- Matplotlib
